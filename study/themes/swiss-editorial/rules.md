# Rules

## Grid system

The grid is the foundation — every element aligns to it:
- 12-column grid with consistent gutters
- Baseline grid for vertical rhythm (8px or 4px unit)
- Asymmetric layouts allowed but always grid-aligned
- No element floats outside the grid without explicit intent

## Typography as hierarchy

Typography does the work that color and decoration do in other themes:
- Size contrast creates hierarchy (minimum 1.5x ratio between levels)
- Weight contrast reinforces (400 body, 700 headings)
- One typeface family preferred, two maximum
- Display type can be oversized — this is the theme's decoration

## Color restraint

- Primary palette: black, white, 2-3 grays
- One accent color maximum (used sparingly for links and CTAs)
- No gradients, no shadows, no glows
- Color communicates meaning, never decoration

## White space

- Generous margins — content should feel like it breathes
- White space is proportional to content importance
- Section breaks use space, not dividers
- Empty space is not wasted space — it is a design element

## Photography and imagery

- Duotone or black-and-white preferred
- Images are always grid-aligned
- Captions use smaller type, precise positioning
- No decorative illustrations — images carry information
